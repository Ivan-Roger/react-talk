# React talk

Clone this repo

```
$ git clone git@github.com:lifebeyondfife/react-talk.git
```

Install the `serve` tool

```
$ npm install serve
```

Serve the presentation by executing

```
$ serve ./react-talk
```

Open a browser to the following address: [http://localhost:3000/](http://localhost:3000/)
